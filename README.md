# 🇨🇮 Babi-Tarif : Frais & Météo Abidjan

**Babi-Tarif** est une application web d'utilité publique conçue pour simplifier le quotidien à Abidjan. Elle permet de comparer les frais de transactions mobiles et d'anticiper l'impact de la météo sur les transports urbains.

---

## 🌟 Fonctionnalités Clés

### 💰 Comparateur de Frais Mobile Money
Calculez instantanément les frais d'envoi et de retrait pour :
* **Wave** (1% de frais standard)
* **Orange Money**
* **MTN MoMo**
*Trouvez l'option la moins chère pour vos transferts en un clic.*

### 🌧️ Alerte Météo "Spécial Pluie"
Grâce à l'intégration de l'API Open-Meteo, l'application détecte en temps réel s'il pleut sur Abidjan. 
* **Impact :** Affiche une alerte automatique sur la hausse probable des tarifs VTC (Yango, Heetch) et taxis compteurs.

### 📱 Interface Mobile-First & PWA
* **Légère :** Fonctionne parfaitement même avec une faible connexion internet.
* **Téléchargeable :** Peut être ajoutée à l'écran d'accueil de votre téléphone pour une utilisation comme une application native.

---

## 🎯 Pourquoi utiliser Babi-Tarif ?
À Abidjan, chaque franc compte. Que vous soyez un commerçant à **Macaci** ou un étudiant au **Plateau**, cet outil vous aide à mieux gérer votre budget transport et vos transactions numériques.

## 🛠️ Installation Technique
Le projet est développé en **HTML5**, **Tailwind CSS** et **JavaScript (ES6+)**.

1. Cloner le dépôt : `git clone https://github.com/TON_NOM/babi-tarif.git`
2. Ouvrir `index.html` dans votre navigateur.
3. Déploiement recommandé via **Vercel** ou **GitHub Pages**.
