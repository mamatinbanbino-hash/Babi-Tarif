# Babi-Tarif
Description du Projet : Babi-Tarif

Babi-Tarif est une solution numérique d'utilité publique conçue pour simplifier la vie quotidienne des citoyens d'Abidjan. Développée par un étudiant passionné de la classe de 1ère A2, cette application web combine intelligence financière et données météorologiques en temps réel.



🌟 Fonctionnalités Clés :

Comparateur de Frais Mobile Money : Calculez instantanément les frais d'envoi et de retrait pour Wave, Orange Money et MTN MoMo. Plus besoin de deviner, trouvez l'option la moins chère pour vos transferts.



Alerte Météo "Spécial Pluie" : Grâce à une intégration API, l'application détecte s'il pleut à Abidjan et vous avertit d'une potentielle hausse des tarifs de transport (Yango, Heetch, Taxis compteurs).



Interface Mobile-First : Une application légère, rapide et intuitive, pensée pour être utilisée sur le terrain, même avec une connexion réduite.



🎯 Pourquoi utiliser Babi-Tarif ?

À Abidjan, chaque franc compte. Que vous soyez un commerçant à Macaci ou un étudiant au Plateau, Babi-Tarif vous aide à mieux gérer votre budget transport et vos transactions numériques.
